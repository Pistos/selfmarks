FAQ

Why is there nothing on this website?

Selfmarks was designed from the start to be personal software. It is intended to help individuals store, manage and search their bookmarks from any place and any device that can access the Internet. Since the bookmarks are not shared, there is nothing to show a guest visiting the site.

To see more, #{A( 'login or register', :href => Rs( :openid ) )}.

How is this better than social bookmarking?

Individuals (or small groups) can install Selfmarks on their own servers, which generally translates into a faster user experience because:

  1. The personal server is geographically closer to the individual.
  2. The personal server is not overloaded with millions of other users whose accounts have nothing to do with the individual.
  3. Selfmarks doesn't have to expend resources or clutter the interface with bookmark sharing features.

How do I register or create an account?

If you have an OpenID, #{A( 'use it', :href => Rs( :openid ) )} to automatically create an account (provided the site administrator has allowed automatic registrations).

If you don't want to use OpenID, then contact the site administrator to ask him/her to manually create a user account for you.
